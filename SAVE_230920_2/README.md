# 세 번째 날 (2)
목표 : 코드 구조 개선
시간 : 4시간                        

미니 프로젝트 세 번째 날 약 8시간의 프로젝트를 진행했다.           
오전 4시간 동안에는 지금까지 작성했던 코드를 class로 구분하여 분할했다.         
ViewMain으로 설정한 main 함수를 포함하고 있는 class가 650줄을 초과하면서         
수정할 코드의 위치를 찾기 어려워졌고 대부분이 if와 while문으로 구성되어 있어서         
정확히 어떤 부분에서 어떤 동작을 하는지 판단하기 어려워졌다.         

따라서 유즈케이스 다이어그램을 참고하여         
하나의 블록을 하나의 class로 구성하여 코드를 개선했다.

첨부한 사진처럼 다이어그램의 한 블록별로 class 파일을 따로 생성했고                 
입력값에 따라 각각의 class를 호출하는 방식으로 코드를 변경했다.              
이렇게 하니 원하는 코드를 찾아갈 때 클래스별로 구분되어 있어서 변경할 부분을 찾기 수월해졌다.

박 준 :  프로젝트 소개문 작성
