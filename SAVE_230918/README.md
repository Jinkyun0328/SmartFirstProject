# 첫째 날        
목표 : 데이터베이스 연동 및 유스케이스 코드 구현          
시간 : 3시간                        

미니 프로젝트 첫 번째 날의 약 3시간 동안 프로젝트를 진행했다.                        
강의 시간에 작성한 구구단 게임을 참고하여                        
while문과 if문으로 유스케이스 다이어그램을 코드로 구현했고                        
DB를 할당받아 JDBC로 자바와 데이터베이스를 연동했다.                        
또한 DTO와 DAO MODEL을 구현하여 회원가입과 로그인 등을 구현했다.                        

## 역할 분류                        
김성민 : 블랙잭, 슬롯머신, 룰렛 등의 게임에 대한 설계 및 알고리즘 모델 설계                        
강채린 : MP3 player를 활용한 효과음 수집 및 모델 설계                        
박 준 :  Console에서 트럼프 카드를 출력하기 위한 그래픽 요소 설계                        
송희주 : DB에 데이터를 전송하기 위한 DAO MODEL 설계                        
최진균 : View 모델 설계 및 유스케이스 코드 구현
