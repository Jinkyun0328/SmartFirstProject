# 두 번째 날 (2)
목표 : MP3 Player 모델 설계
시간 : 4시간                        

미니 프로젝트 두 번째 날 약 8시간의 프로젝트를 진행하면서 오후 4시간동안 MP3 Player 모델을 설계했다.   
주제에 맞춰서 효과음은 '게임보너스', '게임스타트', '룰렛', '슬롯머신' 등의 효과음을    
무료 효과음 사이트에서 검색하여 다운로드 받아 사용했다.   
참고) https://www.mewpot.com/search/sound-effects?is_free=true   
   
강의시간에 배운 MP3 Player 모델과 controller 중에서 프로젝트에서는 이전곡과 다음곡 재생이 필요없기 때문에      
제목을 입력하면 해당하는 음악이 출력되도록 컨트롤러를 설계했다.       
각 상황에 맞춰서 어떤 효과음을 넣을지 설계하여 게임을 진행하는 동안 다채로운 효과음이 들리도록 하였다.
