# SmartFirstProject
## 프로젝트 개요
![images1](https://user-images.githubusercontent.com/123911778/269862986-61a7269d-3862-4a1d-80da-0757a6433b28.jpg)
주제 : 데이터베이스를 활용한 카지노 게임
- 회원가입, 로그인, 회원탈퇴 서비스 제공
- 돈을 기준으로 한 개인조회와 전체조회 서비스 제공
- 블랙잭, 룰렛, 슬롯머신의 게임 서비스 제공
