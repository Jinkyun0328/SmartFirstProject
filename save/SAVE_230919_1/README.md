# 두 번째 날 (1)
목표 : 게임 알고리즘 설계   
시간 : 4시간                        

미니 프로젝트 두 번째 날 약 8시간의 프로젝트를 진행하면서 오전 4시간동안 게임 알고리즘을 설계했다.
주제인 카지노에 맞춰서 게임은 슬롯머신, 블랙잭, 룰렛으로 총 3가지를 준비했다.
게임의 규칙을 세세하게 분석하여 알고리즘을 구성하였고
사용자가 베팅금과 답을 입력하여 랜덤하게 정해진 답을 맞추면 돈을 획득하고 틀리면 잃게 된다.
지난 날 작성했던 DAO 모델을 사용하여 돈을 얻고 잃었을 때 데이터베이스에 저장된 money를 update할 수 있게 한다.

## 역할 분류                        
김성민 : 블랙잭, 슬롯머신, 룰렛 등의 게임에 대한 설계 및 알고리즘 모델 설계                        
강채린 : MP3 player를 활용한 효과음 수집 및 모델 설계                        
박 준 :  예비군 훈련으로 인한 부재                        
송희주 : 카드 및 룰렛 등의 그래픽 설계                        
최진균 : View 모델 설계 및 코드 종합
